Trade Bitcoin with Simulated money

Simulated Bitcoin cryptocurrency trading platform that enables users with little to no trading experience to engage in risk-free trading using virtual funds. The real-time Bitcoin price and candlestick chart data are sourced from the Binance API, providing users with the ability to visualize the dynamic Bitcoin graph in real-time. Drawing inspiration from platforms like Binance and TradingView, this project is designed to encourage individuals who may be hesitant or fearful of trading to gain hands-on experience in a safe and supportive environment.

What's tools this project used?

Binance API

Utilizing the BTCUSDT JSON data from the Binance API to reflect the real-time price of Bitcoin.

For more detials

TradingView Chart

Utilizing the TradingView Candlestick chart sourced from the TradingView github, seamlessly integrated with the Binance API and configured with a 5-minute time interval.

Chart.js

Employing the Chart.js JavaScript library for crafting the dashboard template, enhancing visualizations and ensuring a clean and user-friendly interface.
